<Panel label={<H3 appearAs={H5}>Arguments</H3>}>

      sending an email or SMS.
    </Parameter>

```C#

string completedAt;

string emailAddress;
string body;
string subject;
string phoneNumber;
string postage;
string reference;
string sentAt;
string status;
Template template;
string type;
string createdByName;
string createdByEmailAddress;
```
