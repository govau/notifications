```javascript
client
  .getNotificationById(id)
  .then(response => console.log(response))
  .catch(err => console.error(err))
```

The method returns a <a target="_blank" href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Using_promises">promise</a>.

<Panel label={<H3 appearAs={H5}>Arguments</H3>}>
<Parameters>
  <Parameter required name="id">
    The ID of the notification. This can be found in the response after sending an email or SMS.
  </Parameter>
</Parameters>
</Panel>

<Panel label={<H3 appearAs={H5}>Response</H3>}>

If the request is successful the promise resolves with an `object`, for example:

```json
{
    "id": "notify_id",
    "body": "Hello Foo",
    "subject": null | "email_subject",
    "reference": "client reference",
    "email_address": "email address sent to",
    "phone_number": "phone number sent to",
    "type": "sms" | "email",
    "status": "current status",
    "template": {
        "version": 1,
        "id": 1,
        "uri": "/template/{id}/{version}"
     },
    "created_by_name": "name of the person who sent the notification if sent manually",
    "created_at": "created at",
    "sent_at": "sent to provider at",
}
```
