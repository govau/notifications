```C#
TemplatePreviewResponse response = client.GenerateTemplatePreview(
    templateId,
    personalisation
);
```

The parameters in the personalisation argument must match the placeholder
fields in the template. The client ignores any extra fields in the method.

<Panel label={<H3 appearAs={H5}>Arguments</H3>}>
  <Parameters>
    <Parameter required name="templateId" kind="string">
      Sign in to Notify and go to the Templates page to find the template
      ID.
    </Parameter>
    <Parameter optional name="personalisation" kind="Dictionary<string, dynamic>">
      If a template has placeholder fields for personalised information such as
      name or application date, you must provide their values in a dictionary.
    </Parameter>
  </Parameters>
</Panel>

<Panel label={<H3 appearAs={H5}>Response</H3>}>

If the request to the client is successful, the client returns a `TemplatePreviewResponse`:

```C#
public string id;
public string type;
public int version;
public string body;
public string subject; // null if a sms message
```

</Panel>
