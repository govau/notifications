```go
resp, err := client.GenerateTemplatePreview(id,
    notify.Personalisation{
        {"name", "Kim"},
        {"day", "Monday"},
        {"colours", []string{"red", "blue"}},
    },
)
```

The parameters in the personalisation argument must match the placeholder fields in the actual template.

<Panel label={<H3 appearAs={H5}>Arguments</H3>}>
  <Parameters>
    <Parameter required name="id" kind="string">
      Sign in to Notify and go to the Templates page to find the template ID.
    </Parameter>
    <Parameter
      optional
      name="personalisation"
      kind="[]struct{ Key string, Value interface{} }"
    >
      If a template has placeholder fields for personalised information such as
      name or application date, you must provide their values in a map.
    </Parameter>
  </Parameters>
</Panel>

<Panel label={<H3 appearAs={H5}>Response</H3>}>

If the request is successful a `TemplatePreview` struct is returned, for example:

```go
resp := TemplatePreview {
	ID: "bfb50d92-100d-4b8b-b559-14fa3b091cda",
	Type: "email",
	Version: 1,
	Subject: "Email subject",
	Body: "Hi John,\r\n\r\nToday is Monday.",
}
```

</Panel>
