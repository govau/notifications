If the request is successful a `Template` struct is returned, for example:

```go
resp := Template{
    ID:        "bfb50d92-100d-4b8b-b559-14fa3b091cda",
    Name:      "Welcome email",
    Type:      "email",

    UpdatedAt: "2019-06-06T23:01:01.406542Z",
    CreatedBy: "creator@example.com",
    Version:   1,
    Subject:   "Welcome to Notify",
    Body:      "Hi ((name)),\r\n\r\nWelcome to Notify. Today is ((day)).",
}
```
