<Panel label={<H3 appearAs={H5}>Arguments</H3>}>
  <Parameters>
    <Parameter required name="templateId" kind="string">
      Sign in to Notify and go to the Templates page to find the template
      ID.
    </Parameter>
    <Parameter required name="version" kind="int">
      The version number of the template.
    </Parameter>
  </Parameters>

If the request to the client is successful, the client returns a `TemplateResponse`:

```C#
public string id;
public string name;
public string type;
public DateTime created_at;
public DateTime? updated_at;
public string created_by;
public int version;
public string body;
public string subject; // null if an sms message
```
